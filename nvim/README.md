# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

Normal Vim keybinds 
A to insert
V - Select
:w - write


Custom
:Opencode (Requrires opencode to be installed, for vibecoders)
:OpencodeSize (resizing)
Space + E (Fuzzy Explorer)
S (normal mode) lighting finder or whatever it's call jumps to the line instantly
Space x2 - File search
Space + Slash - Grep search
dd doesn't cut it only deletes
yy yanks no need to add the y"+ flag
:wall 

if you don't know vimtut also if you press space bar it will come up with suggestion menus.
